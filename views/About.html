About

Working with other people's code can suck. Especially if it's not well documented or has a good amount of obscure hacks. Sometimes even if the code is great, it can still suck. Trying to explain where a variable is, how some bit of code works, or trying to smash a bug with a fellow developer can suck. That's why I made this site, to provide a quick and easy way for developers to cooperate on the same code segments in real time.

Oh, and I also LOVE to learn new tech (along with expanding on the old) and this site provided an excellent experience to learn.
